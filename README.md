# scribble-notes

**Scribble**

A lightweight notes app built for students — _write it before you forget_

**Why Scribble?**

Most note-taking apps are built for professionals — too complex, too cluttered, too slow to open when a lecturer is already three slides ahead.

**Scribble is built for students.**

Whether you're in a lecture trying to keep up, writing down an assignment brief, or organizing your project ideas — Scribble gets out of your way and lets you just write.

**Features**

- Rich Text Editing — Bold, italic, underline with one click

- Text Colors — 6 colors to color-code your notes
  
- Highlighter — 5 highlight colors, with selective remove
  
- Plain & Lined Canvas — Switch between blank or ruled page
  
- Stickers — Drop and drag emoji stickers anywhere on the canvas
  
- Draw Tools — Pencil, marker, and crayon modes
  
- Save & Search — Save multiple notes, search by title or content
  
- Persistent Storage — Notes stay saved even after closing the browser
  
- Time & Date — Auto-updating clock on every note session
  
**Built With**
  
- HTML5
  
- CSS
  
- Vanilla JavaScript
  
- No frameworks. No dependencies.
  
**Getting Started:**

Open index.html in any browser — that's it.

⚠️ Desktop Only — Scribble is optimized for desktop browsers. Mobile view is not supported yet.

**Design Process:**

Scribble went through 14 iterations — from a rough first layout to a polished, fully functional app. Each version addressed real usability issues: highlight colors showing correctly, sticker drag working on touch, lined pattern aligning with text, notes persisting across sessions.

The design follows a warm terracotta palette intentionally — it feels like paper, not a screen. The goal was an interface that feels calm and familiar, not clinical.

Note on Storage: Scribble uses browser localStorage to save notes — no server or database required. Notes persist on the same device and browser, making it fully functional as a standalone client-side app

**Who It's For?**

- _Students taking lecture notes quickly_

- _Anyone who thinks better by writing_

- _Learners who want to jot down ideas without switching apps_
